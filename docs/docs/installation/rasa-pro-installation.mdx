## Rasa Pro setup


### Install rasa-pro component

```bash
rasa install pro
```

### Add your license information

```bash
rasa pro add-license <your license string>
```

### Add Rasa Pro componenents


```bash
rasa install tracing etl concurrent-redis-ls
```
