Tracker
=======

.. edit-link::

Trackers maintain the state of the a dialogue and can be featurized for machine
learning algorithms right out of the box. The dialogue between the assistant and the
user is represented by a conversation session. To learn more about how to configure the
session behaviour, check out the docs on :ref:`session_config`.
